## naeos marketplace

Browse and install templates, profiles, and plugins

### Synopsis

NAEOS Marketplace for templates, profiles, and plugins.

Example:
  naeos marketplace search "web api"
  naeos marketplace install web-api-template
  naeos marketplace profile list
  naeos marketplace plugin list

### Options

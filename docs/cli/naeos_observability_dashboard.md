## naeos observability dashboard

Start the observability dashboard

### Synopsis

Start a local web dashboard for viewing traces, logs, and metrics.

Example:
  naeos observability dashboard
  naeos observability dashboard --port 9090

```
naeos observability dashboard [flags]
```

### Options

```
  -h, --help       help for dashboard
  -p, --port int   dashboard port (default 9090)
```

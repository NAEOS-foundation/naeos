## naeos plugin init

Scaffold a new plugin project

### Synopsis

Scaffold a complete plugin project with Go module, SDK boilerplate, tests, Makefile, and GitHub Actions CI.

```
naeos plugin init [name] [flags]
```

### Options

```
  -a, --author string   plugin author
  -d, --desc string     plugin description
  -h, --help            help for init
  -m, --module string   Go module path
```

### Options inherited from parent commands

```
      --dry-run                global dry-run mode: preview without writing to disk
      --output-format string   output format: json, yaml, table (default "table")
      --plugin-dir string      plugin directory (default "/home/codespace/.naeos/plugins")
      --verbose                enable verbose logging
```

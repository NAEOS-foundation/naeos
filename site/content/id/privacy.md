## Informasi yang Kami Kumpulkan

NAEOS adalah alat command-line yang berjalan secara lokal di mesin Anda. Kami tidak mengumpulkan atau mentransmisikan data apa pun dari penggunaan alat CLI NAEOS itu sendiri.

Saat Anda mengunjungi situs web NAEOS (naeos.dev), kami dapat mengumpulkan:

- **Data penggunaan anonim** melalui analitik yang berfokus pada privasi (tanpa data pribadi)
- **Data API GitHub** — saat Anda mengunjungi halaman yang menampilkan statistik GitHub (bintang, fork), browser Anda mengambil data dari API GitHub publik. Tidak ada data autentikasi yang dikumpulkan.

## Data yang TIDAK Kami Kumpulkan

- NAEOS tidak mengumpulkan telemetri dari alat CLI
- NAEOS tidak melacak spesifikasi, proyek, atau kode yang dihasilkan
- NAEOS tidak menggunakan cookie untuk tujuan pelacakan (hanya flag penyimpanan lokal untuk persetujuan cookie)

## Kontak

Untuk pertanyaan terkait privasi, silakan buka issue di [repositori GitHub NAEOS](https://github.com/NAEOS-foundation/naeos).

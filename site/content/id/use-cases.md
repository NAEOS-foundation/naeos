## Arsitektur Microservices

Desain, hasilkan, dan deploy microservices dengan NAEOS. Tentukan layanan, API, dependensi, dan target deployment dalam satu spesifikasi.

- Hasilkan kode layanan Go atau TypeScript dengan endpoint REST
- Buat Dockerfiles dan konfigurasi docker-compose
- Ekspor manifes Kubernetes
- Hasilkan dokumentasi API secara otomatis

**Stack tipikal:** Go, TypeScript, Docker, Kubernetes

---

## Aplikasi Serverless

Bangun aplikasi serverless dengan arsitektur event-driven. Tentukan fungsi Lambda, sumber event, dan sumber daya cloud.

- Hasilkan kode handler Python atau TypeScript
- Buat konfigurasi serverless framework
- Ekspor kebijakan IAM dan definisi sumber daya cloud
- Deployment multi-environment (dev, staging, production)

**Stack tipikal:** Python, TypeScript, AWS Lambda, Serverless Framework

---

## Telusuri di Semua Bahasa

Kelola codebase polyglot dari satu spesifikasi. NAEOS menghasilkan kode yang konsisten di semua bahasa.

- Hasilkan backend Go, frontend TypeScript, layanan data Python
- Struktur proyek yang konsisten di semua bahasa
- Spesifikasi bersama untuk kontrak lintas-bahasa
- Pipeline generasi terpadu

**Bahasa yang didukung:** Go, TypeScript, Python, Java, Rust

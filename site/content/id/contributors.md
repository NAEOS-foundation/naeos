## Cara Berkontribusi

Kami menyambut kontribusi dari berbagai jenis. Berikut beberapa cara Anda dapat membantu:

- **Kode** — Kirim pull request untuk perbaikan bug atau fitur baru
- **Dokumentasi** — Perbaiki docs, tulis tutorial, atau terjemahkan konten
- **Issues** — Laporkan bug atau sarankan fitur melalui GitHub Issues
- **Komunitas** — Bantu menjawab pertanyaan di GitHub Discussions
- **Desain** — Perbaiki situs web, logo, atau branding

Apa itu NAEOS?

NAEOS (Nusantara Engineering & Architecture Operating System) bukan sekadar generator proyek. NAEOS adalah runtime rekayasa yang memahami spesifikasi, membangun model internal (NEIR), mengatur rencana eksekusi, menghasilkan artefak, memvalidasi hasil, dan menjaga proyek tetap selaras dengan spesifikasi sepanjang siklus hidupnya.

From a single YAML specification, NAEOS generates complete project structures, code, documentation, CI/CD pipelines, deployment manifests, and AI context — across 5 programming languages and 6 AI platforms.

Fitur Utama

Fitur Utama — 35+ CLI commands, 5 languages, 6 AI platforms, 56+ specifications.

Pipeline Engine

Pipeline DAG 9-tahap dari parsing hingga ekspor. Parse, normalisasi, resolusi, bangun, validasi, jadwalkan, hasilkan, kompilasi, dan ekspor — dengan observabilitas penuh di setiap tahap.

Kompiler AI

Ubah NEIR menjadi set instruksi AI untuk 6 platform: GitHub Copilot, Claude Code, Cursor, Gemini CLI, Codex, dan OpenCode.

Generator Kode

Hasilkan kode siap-produksi dalam Go, TypeScript, Python, Java, dan Rust. Berbasis template dengan adapter per-bahasa.

Model NEIR

NAEOS Engineering Intermediate Representation — model kanonikal yang mendeskripsikan seluruh sistem: proyek, arsitektur, modul, layanan, API, infrastruktur, dan keamanan.

Tata Kelola & Kebijakan

Mesin kebijakan bawaan dengan 7 operator, RBAC, jejak audit, dan tinjauan artefak. Jaga tim Anda tetap selaras dengan aturan organisasi.

Marketplace

Publikasikan, temukan, dan pasang profil industri serta plugin. 5 profil bawaan: SaaS, AI Agent, FinTech, Healthcare, Government.

Mulai Cepat

Mulai Cepat — from spec to code in 4 steps.

1

Install NAEOS

Install via Go, Docker, or download the binary.

bash
go install github.com/NAEOS-foundation/naeos/cmd/naeos@latest
2

Create a Specification

Define your project in YAML or JSON.

yaml
project: my-app
modules:
  - name: auth
    path: ./auth
  - name: api
    path: ./api
    dependencies: [auth]
services:
  - name: gateway
    kind: http
    port: 8080
generation:
  languages: [go, typescript]
3

Run the Pipeline

Execute the full pipeline — parse, validate, generate, and export.

bash
naeos run --input spec.yaml --output-dir ./out
4

Compile for AI Assistance

Generate AI instruction sets for Copilot, Claude, Cursor, and more.

bash
naeos compile --all --input spec.yaml

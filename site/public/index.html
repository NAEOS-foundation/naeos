What is NAEOS?

NAEOS (Nusantara Engineering & Architecture Operating System) is not just a project generator. It is an engineering runtime that understands specifications, builds an internal model (NEIR), orchestrates execution plans, generates artifacts, validates results, and keeps projects aligned with specifications throughout their lifecycle.

From a single YAML specification, NAEOS generates complete project structures, code, documentation, CI/CD pipelines, deployment manifests, and AI context — across 5 programming languages and 6 AI platforms.

Read the Docs

Core Features

Core Features — 35+ CLI commands, 5 languages, 6 AI platforms, 56+ specifications.

Pipeline Engine

9-stage DAG pipeline from parsing to export. Parse, normalize, resolve, build, validate, schedule, generate, compile, and export — with full observability at every stage.

AI Compiler

Transform NEIR into AI instruction sets for 6 platforms: GitHub Copilot, Claude Code, Cursor, Gemini CLI, Codex, and OpenCode.

Code Generator

Generate production-ready code in Go, TypeScript, Python, Java, and Rust. Template-driven with per-language adapters.

NEIR Model

NAEOS Engineering Intermediate Representation — a canonical model that describes the entire system: project, architecture, modules, services, APIs, infrastructure, and security.

Governance & Policy

Built-in policy engine with 7 operators, RBAC, audit trails, and artifact review. Keep your team aligned with organizational rules.

Marketplace

Publish, discover, and install industry profiles and plugins. 5 built-in profiles: SaaS, AI Agent, FinTech, Healthcare, Government.

Quick Start

Quick Start — from spec to code in 4 steps.

1

Install NAEOS

Install via Go, Docker, or download the binary.

bash
go install github.com/NAEOS-foundation/naeos/cmd/naeos@latest
2

Create a Specification

Define your project in YAML or JSON.

yaml
project: my-app
modules:
  - name: auth
    path: ./auth
  - name: api
    path: ./api
    dependencies: [auth]
services:
  - name: gateway
    kind: http
    port: 8080
generation:
  languages: [go, typescript]
3

Run the Pipeline

Execute the full pipeline — parse, validate, generate, and export.

bash
naeos run --input spec.yaml --output-dir ./out
4

Compile for AI Assistance

Generate AI instruction sets for Copilot, Claude, Cursor, and more.

bash
naeos compile --all --input spec.yaml
